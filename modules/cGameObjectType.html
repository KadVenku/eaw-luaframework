Module cGameObjectType

GetCategoryMask ()
Returns the cGameObjectType's CategoryMask. Concatenates all tagged category masks with the '|' operator for filtering purposes.

Returns:

    string

Usage:

    Consider this example table for the CategoryMask attribute:
self.CategoryMask = { "Capital", "Structure", "SpaceHero" }
self:GetCategoryMask()
This call would then return:
"Capital | Structure | SpaceHero"

GetInstancesOfSelfFilteredBy (filterType, filterObject)
Returns a list of objects filtered by the criteria specified. Possible filters are (Might be expanded later on): "PLANET": Returns all instances of the cGameObjectType filtered by the planet they reside on. "OWNER": Returns all instances of the cGameObjectType filtered by the owner.

Parameters:

  • filterType string String filter for the function.
  • filterObject luaObject object to filter by.

Returns:

    table If not objects can be found it returns false instead.
