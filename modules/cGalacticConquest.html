Initialisation

New (newGCName)
Constructor. Creates a new LUA object of type cGalacticConquest.

Parameters:

  • newGCName String Used as key to load all relevant data from the static conquest table.
Initialise ()
Initialises the galactic conquest scenario. This function creates all necessary objects and is responsible for linking them correctly after creation. This has to be called after the constructor, as some object might not yet be valid.
CreatePlanets ()
Initialises the planets loaded from the active planet list and registers them with the GC.
InitialisePlanets ()
Initialises the object relations for planets. Setting up the planetary ownership and things alike.
CreateFactions ()
Initialises the factions loaded from the active faction list and registers them with the GC.
InitialiseFactions ()
Initialises the object relations for factions. Setting up the faction relations and things alike.
InitialiseTradeRoutes ()
Initialises all registered trade routes.
