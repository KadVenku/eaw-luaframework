Functions

InitialiseTraits (traitTable)
Initializes the faction traits. Only loads traits which are defined, if a trait is not defined, the trait will be skipped.

Parameters:

  • traitTable
GetTraits ()
Returns a faction's traits. Traits are keywords used to describe a faction, they influence the base behaviour and buff/debuff values.

Returns:

    table Trait table.
GetRawMilitaryStrength ()
Returns a faction's raw military strength without trait modifiers applied. This should not be used unless a specific function requires the raw value.

Returns:

    number Raw military strength.
GetHomeSector ()
Returns the home sectors of the planets.
EvaluateRawMilitaryStrength ()
Evaluates a faction's raw military strength. This should only be called before the raw strength is needed as a value, not continuously updated without use.

Returns:

    number Raw strength.

Initialisation

New (parentGC, newFactionName)
{abstract} Constructor. Creates a new LUA object of type cFaction.

Parameters:

  • parentGC cGalacticConquest The parent GC for this faction.
  • newFactionName String Used as key to load all relevant data from the static faction table.
InitialiseHomeSector (sectorPlanetTable, capitalPlanetName)
Initialises the home sector of a faction. Attributes the specified planets to the faction. Updates a planet to be capital.

Parameters:

  • sectorPlanetTable table Contains all planets of a sector.
  • capitalPlanetName string Name of the capital. Has to be contained within the sector table.
