Module cGUIElement

{abstract} cGUIelement base class.

All cGUIElement_[...] inherit the functionality from this base class.

Functions

Initialise () {abstract} Initialises the data required for the button (like story events, flags, etc).
IsFlaggedAs (flagObject) Tests whether the GUIelement is flagged with a certain flag/list of flags.
Lock () Locks the GUI element.
Unlock () Unlocks the GUI element.
ToggleLockState () If the GUI element is locked, it'll be unlocked and vice versa.
Execute () {abstract} Executes the functionality associated with the GUI element.
