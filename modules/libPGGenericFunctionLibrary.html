Module libPGGenericFunctionLibrary

Generic function library.

Provides access to functions which are being reused all over the framework and are of general use throughout the mod.

IsObjectTaggedAs (tagTable, tagObject)
Tests whether a given object is tagged with a certain flag. LUA objects are often tagged with flags in the mod, this generic function allows for testing for a specific flag or a specific set of flags.

Parameters:

  • tagTable table The list of tags an object is tagged with. Usually a self-contained attribute.
  • tagObject table/string Flag the object should be tagged with.

Returns:

    bool

AddPadding (origVal, strLen, paddingChar)
Applies padding to the given value to a given size. The function will pad or cut a given string, depending on its length in relation to the given padding size.

Parameters:

  • origVal string Original string.
  • strLen number The length the string will be padded to.
  • paddingChar string The padding char. If not provided it'll default to "0"
