Module libDebugFunctions

Generic debugging functionality.

Provides threaded functions to push messages to the droid log or the mission holochron.

Debugging Functions

debugPushDroidMessage (msgContent) Pushes a message to the droid log.
debugPlanetOwnerChange (planetStringKey, prevOwnerStringKey, newOwnerStringKey) Displays a planet owner change message via the mission holochron.
debugPlanetSelected (planetStringKey, gObjectTable, sObjectTable) Displays a planet selected message via the mission holochron.
debugEventExecuted (eventType, isTimed, eventTimer) Displays a (timed) event executuion message via the mission holochron.
debugGUIElementLockUnlockEvent (guiElemStringKey, evtType) Displays a GUI element (un-)locked message via the mission holochron.
debugAvailableObjectType (playerFactionKey, objectStringTable) Displays a list of available buildings for a given faction via the mission holochron.
debugPlanetContested (planetStringKey) Displays a message if an enemy fleet is orbiting a planet.
debugFactionRawMilitaryStrength (factionStringKey, factionStrength) Displays a faction's raw military strength.

Threaded Output Handler

tOutPushDroidMessage (msgContent) Threaded output for the droid log push.
tOutPlanetOwnerChange (table) Threaded output for the planet owner change push.
tOutPlanetSelected (table) Threaded output for the planet selected message push.
tOutEventExecuted (table) Threaded output (timed) event executuion message push.
tOutGUIElementLockUnlockEvent (table) Threaded output for the GUI element (un-)locked message push.
tOutAvailableObjectType (table) Threaded output for the available buildings message push.
tOutPlanetContested (planetStringKey) Threaded output for the planet selected message push.
tOutFactionRawMilitaryStrength (table) Threaded output for the faction's raw military strength push.
tOutFoundSuitablePlanetInReach (table) Threaded output for the faction's raw military strength push.
tOutRandomPlanetInReach (table) Threaded output for the faction's raw military strength push.
